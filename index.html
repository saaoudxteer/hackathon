Bienvenue sur la constellation de l'AMU Hackathon-2026. Chaque point lumineux de ce réseau représente la contribution d'un étudiant ou d'une équipe à un projet commun.

Comment participer ?

  • Forker le dépôt Slaytt/Hackathon-2026.
  • Créer une branche et ajouter votre projet dans le dossier noeud du dépôt principal.
  • Ajouter les informations de votre projet dans le fichier registry.json en respectant le format.
  • Choisissez votre propre couleur via le champ "color": "#HEX".
  • Nous reviewerons votre projet avant de le merger.

Le graphe se mettra à jour automatiquement !
